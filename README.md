🎬 Smart Movie Recommender System
📌 Project Overview

This project is a Smart Movie Recommender System that suggests movies based on user input using Content-Based Filtering.
It analyzes movie features such as genres and overview to recommend similar movies.

🚀 Features
🔍 Search any movie by title
🎯 Get top similar movie recommendations
⚡ Fast and efficient recommendation system
🧠 Uses Machine Learning techniques
🌐 Simple web interface using Streamlit
